# Text Analyzer Tool 📊

A simple yet powerful Python script to analyze text input. This project was developed to practice string manipulation, loops, and basic data processing in Python.

## Features
*   **Text Cleaning:** Automatically removes punctuation (`, . ; : ¡ ! ¿ ?`).
*   **Character Counter:** Tracks the frequency of specific letters chosen by the user.
*   **Word Analytics:** Counts the total number of words and identifies the first and last characters.
*   **Keyword Search:** Specifically checks if the word "Python" is present in the text.
*   **User Experience:** Includes timed delays to simulate a processing interface.

## How to use
1. Clone this repository:
   git clone https://github.com/jerocubaque12/Text-Analyzer-tool.git
2. Run the script:
   python main.py
3. Follow the on-screen instructions to enter your text and the letters you want to analyze.

## Requirements
*   Python 3.6+
*   `time` module (built-in)
